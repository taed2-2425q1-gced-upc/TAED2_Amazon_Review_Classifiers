1""" 

2This module sets up project paths and configurations by loading  

3environment variables from a .env file. It also configures logging using the  

4Loguru library and integrates progress bar support with tqdm if available. 

5 

6The module defines the following directory paths for organizing project files: 

7- PROJ_ROOT: Root directory of the project. 

8- DATA_DIR: Directory for storing data. 

9- RAW_DATA_DIR: Subdirectory for raw data. 

10- INTERIM_DATA_DIR: Subdirectory for interim data. 

11- PROCESSED_DATA_DIR: Subdirectory for processed data. 

12- EXTERNAL_DATA_DIR: Subdirectory for external data. 

13- MODELS_DIR: Directory for storing machine learning models. 

14- RESOURCES_DIR: Directory for storing additional resources. 

15- REPORTS_DIR: Directory for generating reports. 

16- FIGURES_DIR: Subdirectory for storing figures related to reports. 

17 

18If tqdm is installed, it configures the logger to support progress bars. 

19""" 

20 

21from pathlib import Path 

22 

23from dotenv import load_dotenv 

24from loguru import logger 

25 

26# Load environment variables from .env file if it exists 

27load_dotenv() 

28 

29# Paths 

30PROJ_ROOT = Path(__file__).resolve().parents[1] 

31logger.info(f"PROJ_ROOT path is: {PROJ_ROOT}") 

32 

33DATA_DIR = PROJ_ROOT / "data" 

34RAW_DATA_DIR = DATA_DIR / "raw" 

35INTERIM_DATA_DIR = DATA_DIR / "interim" 

36PROCESSED_DATA_DIR = DATA_DIR / "processed" 

37EXTERNAL_DATA_DIR = DATA_DIR / "external" 

38 

39MODELS_DIR = PROJ_ROOT / "models" 

40 

41RESOURCES_DIR = PROJ_ROOT / "resources" 

42REPORTS_DIR = PROJ_ROOT / "reports" 

43FIGURES_DIR = REPORTS_DIR / "figures" 

44 

45# If tqdm is installed, configure loguru with tqdm.write

47try: 

48 from tqdm import tqdm

50 logger.remove(0)

53 pass
